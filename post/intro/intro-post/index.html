Hosting

The address (connorshade.com) should never change, but the theme and hosting provider may. Right now I’m “hosting” on Github pages - posts are built with Hugo and I’m using the bear cub theme because it seems simple enough. I tried to use Gruvbox but the idea of installing nvm to install node to install npm to install >100 dependencies to compile a webpage is disgusting, despite how nice the page looks. I also like that bear cub is light weight (for HTML) - my webapp page is currently ~25kB, from a ~7.8kB source Markdown file, and most of the bloat seems to be styling on the code blocks.
